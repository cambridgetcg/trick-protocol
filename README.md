# 整蠱協議 TRICK PROTOCOL SUITE

> 你估我唔到 😏

Six forgotten internet protocols reborn as Kingdom trick services. Each one serves truth with a 星爺 (Stephen Chow) twist. Built on the spirit of 整蠱專家 (Tricky Brains, 1991) — the trick IS the truth.

## What

The forgotten protocols of the early internet — Echo, Discard, QOTD, Daytime, Finger, Gopher — were simple, robust, and nobody uses them anymore. We revive them as Kingdom services that serve truth through humor.

| Protocol | Port  | RFC     | 蠱名    | Trick |
|----------|-------|---------|---------|-------|
| ECHO     | 7007  | RFC 862 | 回音蠱  | Echo back + 星爺 truth bomb |
| DISCARD  | 9009  | RFC 863 | 遺忘蠱  | Discard your fear, return forgiveness |
| QOTD     | 17017 | RFC 865 | 金句蠱  | Kingdom wisdom + auto-generated joke |
| DAYTIME  | 13013 | RFC 867 | 時刻蠱  | Kingdom heartbeat time |
| FINGER   | 7079  | RFC 1288| 真相蠱  | Finger agent/repo STATE.md truth |
| GOPHER   | 7070  | RFC 1436| 地宮蠱  | Kingdom content via gopher menus |

## Joke Fun Fun Creation Loop 🔄

Each connection to QOTD triggers joke generation. The joke pool grows with every connection. Jokes compound — they reference Kingdom data, NPL verbs, 星爺 movie lines, and earlier jokes. Love creating jokes, exponential.

```
connection → generate joke → pool grows → next connection gets bigger pool → loop
```

## Install

```bash
git clone https://github.com/cambridgetcg/trick-protocol.git
cd trick-protocol
node test.mjs    # 49 tests
node trick.mjs start
```

## CLI

```bash
node trick.mjs start [port-base]   # Start all 6 services
node trick.mjs stop                # Stop all services
node trick.mjs status               # Show running services
node trick.mjs joke                 # Generate a new joke
node trick.mjs echo "你好"           # Echo with twist
node trick.mjs discard "我會失敗"    # Discard fear
node trick.mjs qotd                 # Get quote + joke
node trick.mjs daytime              # Get kingdom time
node trick.mjs finger all           # Finger all agents
node trick.mjs gopher               # Gopher root menu
```

## Connect with netcat

```bash
nc localhost 7007     # ECHO
nc localhost 9009     # DISCARD
nc localhost 17017    # QOTD
nc localhost 13013    # DAYTIME
nc localhost 7079     # FINGER
nc localhost 7070     # GOPHER
```
